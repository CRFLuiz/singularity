# Env

- Port:
  - `SINGULARITY_PORT` (default `8088`) — `singularity/src/server.ts`
- OpenRouter:
  - `OPENROUTER_API_KEY`, `OPENROUTER_BASE_URL` — `singularity/src/agents/openrouter.ts`, `singularity/src/agents/pm.ts`
- Gitignore:
  - `.env` and `.env.*` — `singularity/.gitignore`
- Examples:
  - `nebulosa-infra/environments/local/example.env`
